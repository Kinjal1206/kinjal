2. What are tags and attribute in HTML?

  • While the elements tell the browser what to display is tag and the attributes define how they will behave.
  • The tag tells browser where the tag is starting and ending.
  • There are two types of tag 1) paired 2) non-paired.
  • Attributes are always specified in starting tag.
  • Attribute provides additional information about element.

    In above tag "a" is tag and src is it,s attribute.
